Character.AI → CrushOn.AI v0.1.1

Export your Character.AI bots as Tavern character cards. One-click import to CrushOn.AI (and any Tavern-compatible platform).

How to use

1. Open a character page on character.ai
Either /chat/<id> or /character/...
2. Two pink/purple buttons appear top-right

Two export formats

📥 Export JSON
Downloads <name>.json — Tavern V1 flat schema. Upload to CrushOn "import existing character".
🎴 Export PNG Card
Downloads <name>.png — avatar image with embedded character data (tEXt/chara chunk). Compatible with CrushOn.AI, SillyTavern, RisuAI, Chub.ai.
Export characters you created, or characters where the creator set Definition Visibility = Public.
